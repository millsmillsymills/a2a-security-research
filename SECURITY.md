# Security Policy

This is a portfolio threat-modeling repository, not production software.

All proof-of-concept code targets only operator-owned, local (`127.0.0.1`)
infrastructure. The PoCs pair every exploit with its mitigation and exist to
demonstrate documented A2A threat classes for defensive education.

Do not point any PoC at infrastructure you do not own. No real credentials or
third-party agents are used.

Report issues by opening a GitHub issue.
